# Algeo02-22125

## Backend

Setup virtual environment

```
virtualenv venv
```

Aktivasi venv

```
source ./venv/Scripts/activate  # Windows
source venv/bin/activate        # WSL / Linux / Mac
```

Deaktivasi venv

```
deactivate
```

Install komponen-komponen di requirement.txt

```
pip install -r requirements.txt
```
